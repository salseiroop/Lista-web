O que achei do Git e GitHub?

Sendo sincero, o começo foi meio tenso! Como formatei meu PC recentemente, o Git não queria me deixar comitar de jeito nenhum porque eu não tinha configurado meu e-mail e nome. Deu um trabalhinho entender os erros do terminal, mas depois que configurei a ID e entendi o fluxo de add, commit e push, ficou mais suave.

O GitHub Pages também é massa porque já coloca o site no ar direto. Achei massa aprender a usar um pouco do git.
